# Instructions

> The following instructions require that Python 3.12.7 is already installed.

Download the repository. Open the terminal in the saved folder and install the requirements:
>  pip install -r requirements.txt
Install the SCIP solver((https://pypi.org/project/PuLP/)[pulp help page]):
>  pip install pulp[scip]
To run the script:
>  python path_planning_final.py
